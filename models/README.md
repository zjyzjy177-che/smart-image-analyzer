# 离线模型目录

本目录随本地部署包提供，程序会优先从这里加载模型，不依赖当前用户的缓存目录。

| 目录 | 用途 | 约占空间 |
|---|---|---:|
| `resnet50/` | ImageNet 图像分类 | 98 MB |
| `blip-image-captioning-base/` | 英文整图描述 | 945 MB |
| `opus-mt-en-zh/` | 整图描述英译中 | 301 MB |
| `easyocr/` | 简体中文、英文文字检测与识别 | 100 MB |

模型总计约 1.44 GB。大权重文件由 Git LFS 管理；完整克隆后需执行
`git lfs pull`。OCR 仅打包简体中文和英文，不包含其他语种。
